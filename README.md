# LiuYun(流云)(WIP)
流云是一个流程引擎，它并不保证遵循BPM规则，但它的目标是足够简单和高扩展，方便定制。 

它现在还在建设中，仅完成少量代码。

所有的使用示例均放置于liuyun-test中。
